# Codehop
## WHAT IS CODEHOP?
**>Codehop is a turing tarpit esoteric programming language.**

## WHAT IS HIS COOL FEATURE?
**>His cool feature is that the language allows you to jump/pass certain instructions to go back or forward.**

## HOW TO INSTALL AND USE CODEHOP?
**CodeHop was written in C++, It reads a file that is called "CodeHopFile.txt" and interprets it. You will have to write your code in this file and launch CodeHop.exe. Make sure that CodeHopFile.txt is in the codehop folder!**
