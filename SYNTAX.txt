# - Used to end a line.                                     
. - Has no effect, only passes to the next statement.
2 - Skip two instructions (The present and the one after).
3 - Skip three instructions.
4 - Skip four isntructions.
5 - Skip five instructions.
6 - Return to the instruction before.
7 - Return to the two instructions before.
8 - Return to the three instructions before.
9 - Return to the four instructions before.
/ - Reset the value of x.
= - Output the value of x.
? - Input the value of x.
+ - Add one to the value of x.
- - Substract one to the value of x.
* - Add the square of x to x.
} - Select the next x.
} - Select the previous x.
$ - Add the ASCII value to the string.
: - Output the string.
\ - Reset the string.
& - Skip two instructions if the value of x equals to 0.
_ - Repeat the same line until x equals to 0.
~ - Repeat the after the first three characters same line until x equals to 0 (useful with a &).
